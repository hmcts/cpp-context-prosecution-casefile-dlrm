# [DD-42991] Default Hearing Time for Unallocated hearings

Jira: https://tools.hmcts.net/jira/browse/DD-42991
Epic: Schema alignment for DLRM
Sprint: DLRM Sprint 60
Story points: 5

## User story
As the **Common Platform hearing-creation process**,
I want **an XHIBIT case with a missing courtroom (and, in one case, a missing hearing time) to still produce a valid Unallocated hearing with all defendants/offences listed**,
so that **incomplete listing data from XHIBIT never blocks a hearing from being created in CP**.

## Background
XHIBIT cases are received into Common Platform and, after passing validation, a hearing is created from the hearing details on the case. This story covers the two cases where the courtroom is absent from the incoming hearing details — with and without a hearing time also being absent — and defines how CP should default the hearing so it still lands as a valid Unallocated hearing.

## Acceptance criteria

- [ ] **AC-001: Create an unallocated hearing when courtroom and time are not provided**
  - Given an XHIBIT case has been received in Common Platform (CP) and has successfully passed validation
  - And the hearing details include a courthouse and a hearing date
  - And the hearing details do not include a courtroom
  - And the hearing details do not include a hearing time
  - When the hearing is created in Common Platform
  - Then the hearing shall be created as an Unallocated hearing
  - And default the hearing time to 10am
  - And list all defendants, offences in the hearing as per the hearing details in the case received

- [ ] **AC-002: Create an unallocated hearing when courtroom is not provided but time is provided**
  - Given an XHIBIT case has been received in Common Platform (CP) and has successfully passed validation
  - And the hearing details include a courthouse, hearing date and a hearing time
  - And the hearing details do not include a courtroom
  - When the hearing is created in Common Platform
  - Then the hearing shall be created as an Unallocated hearing
  - And list all defendants, offences in the hearing as per the hearing details in the case received
